  • Wiring/Connectors from/to Exhaust Gas Recirculation (EGR) Potentiometer (G212) faulty
  • Exhaust Gas Recirculation (EGR) Potentiometer (G212) faulty

Possible Solutions

  • Check Exhaust Gas Recirculation (EGR) Potentiometer (G212)
