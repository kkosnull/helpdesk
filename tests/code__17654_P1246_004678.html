17654/P1246/004678 - Needle Lift Sensor (G80): Implausible Signal

Possible Symptoms

  • Malfunction Indicator Light (MIL) ON
  • Engine misfiring/juddering/running rough
  • Engine shutting down

Possible Causes

  • Fuel Supply interrupted/faulty
    • Fuel Tank (temporary) empty
    • Fuel Filter blocked
    • Air in Fuel System
  • Wiring and/or Connector(s) from/to Needle Lift Sensor (G80) faulty
  • Needle Lift Sensor (G80) faulty
  • Injection Pump faulty

Possible Solutions

  • Check Fuel Supply
    • Check Fuel Level
    • Check/Replace Fuel Filter
    • Check for Air in Fuel System
  • Check Wiring and/or Connector(s) from/to Needle Lift Sensor (G80)
  • Check/Replace Needle Lift Sensor (G80)
  • Check Injection Pump

Special Notes

  • Might by stored with other codes like 17656/P1248/004680.
