17958/P1550/005456 - Boost Pressure Control Valve (N75): Control Deviation

Possible Symptoms

  • Malfunction Indicator Light (MIL) ON
  • Engine misfiring/juddering/running rough
  • Power Loss / Limp Mode

Possible Causes

  • Hoses/Pipes incorrectly connected, disconnected, blocked or leaking
  • Charger Pressure Control faulty
    • Solenoid Valve for Boost Pressure Control (N75) faulty
    • Turbocharger faulty
    • Vacuum Unit for Turbocharger faulty

Possible Solutions

  • Check Hoses/Pipes to/between Components
  • Check Charge Pressure Control
    • Check Solenoid Valve for Boost Pressure Control (N75)
    • Check Turbocharger
    • Check Vacuum Unit for Turbocharger
