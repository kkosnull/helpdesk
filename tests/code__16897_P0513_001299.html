16897/P0513/001299 - Immobilizer Incorrect Key

Possible Symptoms

  • Malfunction Indicator Light (MIL) ON

Possible Causes

  • Immobilizer Scan Protection active

Possible Solutions

  • Let the Lock Time (10 Minutes) run down with Ignition (Terminal 15) ON

Special Notes

  • This Fault Code may be stored after the Battery was fully drained and indicates an issue while checking if the Engine is allowed to start. It is very unlikely that either adapting the Immobilizer or matching Keys will will fix this Scenario. For VW Golf (1K) or similar, check TPI 2017171 (RoW) for further Details.
