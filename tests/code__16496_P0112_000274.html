16496/P0112/000274 - Intake Air Temperature Sensor (G42): Signal too Low

Possible Symptoms

  • Malfunction Indicator Light (MIL) ON

Possible Causes

  • Fuse(s) faulty
  • Wiring and/or Connector(s) from/to Intake Air Temperature Sensor (G42) faulty
  • Intake Air Temperature Sensor (G42) faulty

Possible Solutions

  • Check Fuse(s)
  • Check Wiring and/or Connector(s) from/to Intake Air Temperature Sensor (G42)
  • Check/Replace Intake Air Temperature Sensor (G42)
    • Check Measuring Value Blocks (MVB)
