16920/P0000/000000 - Undocumented/Reserved Fault Code

Possible Symptoms

  • Malfunction Indicator Light (MIL) ON

Possible Causes

  • Engine Control Module (ECM) Software faulty
  • Engine Control Module (ECM) Software modified/tuned

Possible Solutions

  • Check/Update Engine Control Module (ECM)
  • Contact Chip Tuner (if applicable)

Special Notes

  • When found in VW Crafter (2E): BJJ/BJK/BJL/BJM Engine, perform Software Update according to TPI 2018060.
  • We've also seen this Fault Code being set after (sometimes delayed) the Engine Control Module (ECM) has been modified by a Tuner. Those Tuners we have spoken to about this knew already or found a way to fix it. ;)
