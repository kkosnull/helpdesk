17653/P1245/004677 - Needle Lift Sensor (G80): Short to Ground

Possible Symptoms

  • Pre-Glow Control Light active
  • Engine running rough
  • Reduced Power output
  • Emissions affected

Possible Causes

  • Needle Lift Sensor (G80) faulty
  • Wiring and/or Connector(s) from/to Needle Lift Sensor (G80) faulty

Possible Solutions

  • Check Needle Lift Sensor (G80)
  • Check Wiring and/or Connector(s) from/to Needle Lift Sensor (G80)
