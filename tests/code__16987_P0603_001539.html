16987/P0603/001539 - Internal Control Module: Non-Volatile Memory Error

Possible Symptoms

  • Malfunction Indicator Light (MIL) active

Possible Causes

  • Control Module faulty
  • Soft Coding for Control Module not correct

Possible Solutions

  • Replace Control Module
  • Code Control Module

Special Note

TCM part number 3B0 927 156 AH will have this fault code when new. Perform soft coding of "0001104" and fault should clear.
