16726/P0342/000834 - Camshaft Position Sensor (G40): Signal Low

Possible Symptoms

  • Malfunction Indicator Light (MIL) active

Possible Causes

  • Wiring from/to Camshaft Position Sensor (G40) faulty
  • Camshaft Position Sensor (G40) faulty
  • Timing misaligned

Possible Solutions

  • Check Wiring from/to Camshaft Position Sensor (G40)
  • Check Camshaft Position Sensor (G40)
  • Check Timing

Special Notes

  • When found in Skoda Fabia (5J) with 1.2l MPI Engine, check Camshaft Position Sensor (G40). See TPI 2020297 (AM 2009/25) for further details and part numbers.
