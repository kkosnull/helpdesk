29709/U1013/053267 - Control Module Not Coded

Possible Symptoms

  • Malfunction Indicator Light (MIL) active

Possible Causes

  • Control Module not coded

Possible Solutions

  • Check Control Module Coding

Special Notes

  • In certain cases this code may be stored even after properly coding a module (e.g. VW RNS510 or 9W7). In such cases make sure to use a proper workshop code (WSC), importer and device number. Entering defaults such as 12345/123/12345 in the boxes beneath the New coding followed by recoding the module to the same value should resolve those issues:
