16507/P0123/000291 - Throttle Position Sensor (G69): Signal too High

Possible Symptoms

  • Malfunction Indicator Light (MIL) ON
  • EPC Light On
  • Increased Idle Speed
  • Bucking at Load Change

Possible Causes

  • Fuse(s) faulty
  • Wiring and/or Connector(s) from/to Throttle Position Sensor (G69) faulty
  • Throttle Position Sensor(s) Faulty
  • Throttle Body dirty
  • Throttle Body faulty

Possible Solutions

  • Check Fuse(s)
  • Check Wiring and/or Connector(s) from/to Throttle Position Sensor (G69)
  • Clean Throttle Body and complete TBA
  • Check Throttle Body Wiring
  • Check Throttle Body
  • Check/Replace Throttle Position Sensor (G69)
    • On DBW Vehicles This Includes Accelerator Pedal Position Sensor 1 (G79) and 2 (G185).
    • On DBC Vehicles This Includes Throttle Position Sensor G69 and G88.

Special Notes

  • See Throttle Body Alignment (TBA)
  • Use Meas. Blocks & Basic Setting for testing
  • When found in the NAR 2.0l TFSI: (BPY, CCTA and CBFA)
    • Check VW TSB 01-09-04 or 2018652 for a wiring repair at the Throttle Body.
    • Check for proper seating at the ignition coils.
  • When found in the NAR Audi 2.0l TFSI, 3.2L FSI and 3.6L FSI engines:
    • Check Audi TSB 97-08-25 or 2018260/4 for a wiring repair at the Throttle Body.
