17087/P0703/001795 - Brake Switch (F): Electrical Malfunction

Possible Symptoms

  • Malfunction Indicator Light (MIL) ON

Possible Causes

  • Wiring and/or Connector(s) from/to Brake Switch (F) faulty
  • Brake Switch (F) faulty

Possible Solutions

  • Check Wiring and/or Connector(s) from/to Brake Switch (F)
  • Check/Replace Brake Switch (F)

Special Notes

  • When found in Vehicles with Drive-by-Cable (DBC) the Brake Switch (F) is connected to the Transmission Control Module. On older Models this Code is stored automatically after Terminal 15/Ignition is switched ON.
