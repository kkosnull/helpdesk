17955/P1547/005447 - Boost Pressure Controle Valve (N75): Short to Ground

Possible Symptoms

  • Irregular behaviour
  • Loss of power
  • Limp mode

Possible Causes

  • Wiring and/or connections faulty (short circuit)
  • N75 faulty

Possible Solutions

  • Check wiring and connections
  • Check / Replace N75
